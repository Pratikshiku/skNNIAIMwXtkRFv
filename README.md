## Student Attendance Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/56ea83fd273d4ff299ac1e3026b70646/ghb20250919) 


### 1. Related Technologies
#### Backend: Java, JavaWeb / Spring, Spring MVC / MyBatis / JDK1.8
#### Frontend: Vue, HTML / CSS / Javascript, etc.
#### Database: MySQL8.0 / MyBatis
### 2. Related Software (Any one of the listed software can be used to run)
#### IDEA

#### Eclipse

#### Visual Studio Code(VScode)

#### Navicat

#### Frontend Environment Node14

### 3. Feature Description
#### The system is divided into (roles): administrators, counselors, students
Main Features:

Personal Center User Data Modification
Secure Logout
Student Account Registration
Teacher Account Registration
Class Management
Announcement Type Management
Attendance Type Management
Course Management
Counselor Management
Student Management
Teacher Management
Early Warning Management
